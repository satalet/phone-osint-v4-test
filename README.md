# 📡 Phone OSINT v4 — مستكشف الأرقام الميداني الشامل

> أداة OSINT مفتوحة المصدر مبنية بـ Python للكشف عن معلومات أرقام الهاتف

---

## ✨ المميزات

| الميزة | التفاصيل |
|---|---|
| 🔍 **OSINT APIs** | Numverify + IPQualityScore للكشف عن الشركة، نوع الخط، مستوى الخطر |
| 📦 **Archive.org** | بحث تلقائي في أرشيف الإنترنت بصيغتين للرقم |
| 🌐 **شبكات التواصل** | روابط بحث لـ Google، Facebook، Instagram، TikTok، LinkedIn، Twitter/X |
| 🗄️ **SQLite** | حفظ تلقائي لكل فحص مع عرض آخر 10 فحوصات |
| 🌍 **متعدد الدول** | دعم خاص لفلسطين، إسرائيل، الأردن، السعودية، مصر، وأكثر |
| ⚙️ **config.json** | حفظ API keys مرة واحدة فقط |

---

## 🚀 التثبيت

```bash
git clone https://github.com/satalet/phone-osint-v4-test.git
cd phone-osint-v4-test
pip install phonenumbers
python3 phone_crawler_v4.py
```
